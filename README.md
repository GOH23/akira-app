## üöÄ Quick Start Guide  
### Prerequisites  
- Node.js v16+  
- Electron-compatible browser (Chrome 90+/Edge recommended)  
- WebGL 2.0 compatible GPU  

### Installation  
```bash
# Clone repository
git clone https://github.com/GOH23/akira-app.git

# Install dependencies
npm install

# Launch Electron app
npm start
```
